# nest
Engine e opensource de busca semântica embarcado desenvolvido para o truw.news Funciona como single-file vector store local /sqlite para embedning local first sem necessidade de infra. Serve para armazenar/comprimir informaçoes para agentics, llms, chunks de aprenizados. Comprime blocos de texto com zstd, FAISS IVF+PQ p/ nearest, metadados, etc
